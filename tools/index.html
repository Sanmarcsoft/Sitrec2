Sitrec Tools

Standalone tools for working with Sitrec data.

LOS CSV Viewer

Interactive 3D visualization tool for Line of Sight (LOS) CSV data exported from Sitrec.

Features:

  • 50km x 50km ground grid (1km divisions) with white lines on grey background
  • Large red north arrow with "N" label at origin
  • 3D track visualization with sensor positions
  • Ground projection lines
  • LOS vectors (every 30th frame) extending to maxRange
  • Interactive orbit controls (left-click to orbit, right-click to pan, scroll to zoom)

PX4 ULog Viewer

Parse and visualize PX4 flight logs (ULog format) in 3D with CSV export capabilities.

Features:

  • Binary ULog parser supporting ULogV1 format
  • 3D flight path visualization with auto-fit camera
  • GPS to local coordinate conversion (ENU frame)
  • Start/end position markers (green/red spheres)
  • Optional velocity vector display
  • Export track data or complete message dump to CSV

Usage

  1. For LOS Viewer: Export LOS data from Sitrec using the "CSV" button on any LOS node
  2. For PX4 Viewer: Obtain a .ulg file from PX4 autopilot or Flight Review
  3. Open the appropriate tool from the list above
  4. Load your file
  5. Interact with the visualization

Technical Details

All tools use a local copy of Three.js r180 (core module + OrbitControls) in the three.js/ directory. This ensures no external CDN dependencies are required.
